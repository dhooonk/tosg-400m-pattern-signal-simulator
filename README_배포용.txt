TOSG Pattern Viewer v1.0
========================

TOSG-400M 패턴 신호 뷰어에 오신 것을 환영합니다!

## 설치 방법

1. 압축 파일을 원하는 위치에 압축 해제합니다
2. `TOSG-Pattern-Viewer.exe`를 더블클릭하여 실행합니다

**별도의 설치 과정이 필요 없습니다!**

## 시스템 요구사항

- **운영체제**: Windows 10 이상 (64비트)
- **메모리**: 최소 4GB RAM (8GB 권장)
- **화면 해상도**: 1280x720 이상
- **저장 공간**: 약 500MB

## 주요 기능

- 📊 타이밍 다이어그램 시각화
- 📈 신호 파형 생성 및 분석
- 💾 Excel 파일로 데이터 내보내기
- 🎨 사용자 정의 신호 설정
- 📁 신호 데이터 저장 및 불러오기

## 사용 방법

1. 프로그램을 실행합니다
2. 좌측 제어판에서 신호 파라미터를 설정합니다
3. "Generate Waveform" 버튼을 클릭하여 파형을 생성합니다
4. 그래프를 확인하고 필요시 Excel로 내보냅니다

자세한 사용 방법은 프로그램 내 도움말을 참조하세요.

## 문제 해결

### 프로그램이 실행되지 않는 경우

**증상**: EXE 파일을 더블클릭해도 아무 반응이 없거나 즉시 종료됨

**해결 방법**:
1. Visual C++ 재배포 패키지를 설치하세요
   - 다운로드: https://aka.ms/vs/17/release/vc_redist.x64.exe
   - 설치 후 컴퓨터를 재시작하세요

2. Windows Defender 또는 백신 프로그램 확인
   - 백신 프로그램이 실행을 차단할 수 있습니다
   - 예외 목록에 TOSG-Pattern-Viewer.exe를 추가하세요

### "DLL을 찾을 수 없습니다" 오류

**해결 방법**:
- Visual C++ 재배포 패키지를 설치하세요 (위 링크 참조)

### 데이터 파일을 찾을 수 없다는 오류

**해결 방법**:
- `config/` 및 `data/` 폴더가 EXE 파일과 같은 위치에 있는지 확인하세요
- 압축 해제 시 폴더 구조가 유지되어야 합니다

### 그래프가 표시되지 않는 경우

**해결 방법**:
1. 프로그램을 완전히 종료하고 다시 시작하세요
2. 화면 해상도가 최소 요구사항을 만족하는지 확인하세요
